Código desnvolvido para uma empresa chamada MGS,onde trata de gerar uma carteirinha médica para os funcionários.
Consiste em quando executar o programa ele vai gerar um menu, no qual irá ter a opção de ´adicionar funcionário`,
´exibir funcionário` ou ´sair`.
Ao adicionar um funcionário ele irá pedir uma série de informações, as quais ficarão salvas no ArrayList.
Ao pedir para exibir funcionários, ele irá no ArrayList e irá printar na tela as informações nele salva.
Ao apertar opção ´3` de sair, ele encerra o funcionamento do programa.
